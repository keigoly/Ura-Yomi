# Chrome Web Store 説明文（日本語）

---

■ ウラヨミ！ってなに？

YouTube動画のコメント欄を、Google Gemini AIの力で丸ごと解析するChrome拡張機能です。
数千件のコメントを瞬時に分析して、視聴者の「本音」を可視化します。


■ できること

【AI解析】
- 全体要約：コメント全体の傾向をAIがわかりやすく要約
- 感情分析：ポジティブ・ニュートラル・ネガティブの割合を円グラフで表示
- トピック抽出：コメント内で話題になっているテーマを自動検出
- 深掘り分析：各感情の代表的コメントをGeminiが選定し、理由とともに提示

【Hidden Gems】
- いいね数は少ないけど、価値のあるコメントをAIが発掘

【コメントビューア】
- YouTube風のスレッド表示
- 人気順・いいね数・投稿日時でソート
- リアルタイム検索
- 「3日前」「2ヶ月前」などの自然な時間表示


■ 対応コンテンツ

- 通常動画（/watch?v=...）
- ショート動画（/shorts/...）
- URLを貼り付けて解析開始もOK
- 最大10,000件のコメントをまとめて解析


■ キャラクターモード

ONにすると、2人のオリジナルキャラクターが解析結果を独自の口調で解説します。
通常の解析結果とは一味違った、エンターテインメント要素を楽しめます。


■ カスタマイズ

- 多言語対応：日本語 / English（UI全体 + AI解析結果も切り替え可能）
- テーマ：ライト / ダークブルー / ブラックの3種類
- フォントサイズ：5段階で調整
- 解析履歴：過去の結果を最大30件保存
- 設定のインポート/エクスポート：JSON形式でバックアップ


■ 使い方（かんたん3ステップ）

1. Googleアカウントで認証
2. YouTube動画ページを開く
3. 解析ボタンをクリック！

APIキーの設定などは不要です。


■ クレジットシステム

- 解析1回あたり3クレジット消費
- 初回登録で無料クレジットをプレゼント
- 追加クレジットは設定画面から購入可能


■ プライバシーについて

- APIキーはサーバー側で管理（ユーザーに公開しません）
- 解析履歴はブラウザ内（localStorage）に保存
- 通信はHTTPS暗号化
- 必要最小限のデータのみ収集
