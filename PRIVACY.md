# プライバシーポリシー

**ウラヨミ！ Powered by Google Gemini**

最終更新日: 2026年2月

## はじめに

「ウラヨミ！ Powered by Google Gemini」（以下「本拡張機能」）は、ユーザーのプライバシーを尊重し、個人情報の保護に努めています。本プライバシーポリシーは、本拡張機能がどのような情報を収集し、どのように使用するかについて説明します。

## 収集する情報

### 1. Google アカウント情報

本拡張機能はGoogle OAuthを使用して認証を行います。認証時に以下の情報を取得します:

- **Google ユーザーID**: アカウントの識別に使用
- **メールアドレス**: アカウント管理に使用
- **表示名**: UI上での表示に使用
- **プロフィール画像URL**: UI上での表示に使用

### 2. YouTube動画データ

解析時に以下のデータをYouTube Data API v3を通じて取得します:

- **動画ID・タイトル**: 解析対象の識別に使用
- **コメントデータ**: テキスト内容、投稿者名、いいね数、投稿日時、返信情報
- **コメント総数**: 表示・解析に使用

### 3. 利用データ

- **クレジット残高**: サービス利用の管理に使用
- **解析履歴**: ユーザーのブラウザ内（localStorage）に最大20件保存
- **設定情報**: テーマ、フォントサイズ、言語設定などのカスタマイズ設定

### 4. 自動的に収集する情報

- **解析リクエストログ**: サーバー側でAPIの利用状況を記録（コスト管理目的）

## 情報の使用目的

収集した情報は以下の目的にのみ使用します:

1. **コメント解析サービスの提供**: YouTube動画コメントのAI分析（感情分析、要約、トピック抽出）
2. **ユーザー認証**: Googleアカウントによるログイン・セッション管理
3. **クレジット管理**: 解析に必要なクレジットの残高管理・課金処理
4. **サービス改善**: APIコスト管理、エラー監視によるサービス品質の維持

## 情報の保存場所

| データ | 保存場所 | 保存期間 |
|-------|---------|---------|
| ユーザーアカウント情報 | サーバーメモリ（インメモリ） | サーバー稼働中のみ |
| セッショントークン | ブラウザ（chrome.storage） | ログアウトまで |
| 解析履歴 | ブラウザ（localStorage） | ユーザーが削除するまで（最大20件） |
| デザイン設定 | ブラウザ（localStorage） | ユーザーが削除するまで |
| 言語設定 | ブラウザ（localStorage） | ユーザーが変更するまで |
| APIコストログ | サーバーメモリ | サーバー稼働中のみ |

## 第三者へのデータ提供

本拡張機能は、以下のGoogle APIサービスを利用しています:

- **Google OAuth 2.0**: ユーザー認証
- **YouTube Data API v3**: コメントデータの取得
- **Google Gemini API**: コメントのAI分析

これらのサービスへのデータ送信は、本拡張機能の機能提供に必要な範囲に限定されます。上記以外の第三者にユーザーの個人情報を販売、取引、またはその他の方法で提供することはありません。

### Google API サービスの利用

本拡張機能のGoogle APIサービスからの情報の使用および転送は、[Google API サービスのユーザーデータポリシー](https://developers.google.com/terms/api-services-user-data-policy)に準拠しています。

## データの安全性

- 認証にはJWT（JSON Web Token）を使用し、セッション情報を安全に管理します
- サーバーとの通信はHTTPSを推奨しています
- APIキーはサーバー側で管理し、クライアントには公開しません
- ユーザーデータは必要最小限のみ収集し、不要なデータは保持しません

## ユーザーの権利

ユーザーは以下の権利を有します:

1. **データの確認**: 設定画面から保存されている解析履歴・設定情報を確認できます
2. **データの削除**: 解析履歴・設定情報はブラウザから削除できます
3. **アカウントの切断**: Googleアカウントの連携はいつでも解除できます
4. **拡張機能の削除**: Chrome拡張機能をアンインストールすることで、ブラウザに保存されたすべてのデータが削除されます

## Cookie の使用

本拡張機能はCookieを使用しません。セッション管理にはJWTトークンとchrome.storage APIを使用しています。

## 子どものプライバシー

本拡張機能は13歳未満の子どもを対象としていません。13歳未満の子どもから意図的に個人情報を収集することはありません。

## プライバシーポリシーの変更

本プライバシーポリシーは予告なく変更される場合があります。変更があった場合は、本ページを更新することでお知らせします。
